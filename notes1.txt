What is Interpolation in Vue js
Interpolation in Vue.js is a fundamental concept used to bind data to the HTML template of your Vue application. 
It allows you to display JavaScript values in the HTML part of your component.
This is done using the "Mustache" syntax (double curly braces {{ }}), 
where you can place a JavaScript expression. Vue.js will automatically update the DOM when the data changes,
making your app reactive.

• What is Two-way binding
Two-way binding in Vue.js is a mechanism that allows properties in the model to be instantly synchronized with the UI ,
thereby ensuring that changes in the user interface reflect immediately in the app's data model and changes in the data model immediately update the UI. 


The "v-model" directive creates a two-way binding on an input, textarea,
or select element (or components that emulate these). It automatically picks the correct way to update the element based on the input type. 
This makes it very useful for creating forms.
