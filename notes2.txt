Parent Component:
Provides data and functionality to its child components through props (properties passed down).
Child Component:
A component that is nested within another component (its parent).
Reuses functionality and UI elements defined in its parent or other components.\

Sibling Components:
Components that share the same parent component but are not nested within each other.
Don't have a direct communication channel by default in Vue.js.
